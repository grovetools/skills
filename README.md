# grove-skills

Agent Skill Integrations

## Installation

```bash
grove install grove-skills
```

## Usage

```bash
grove-skills --help
```

## Development

### Building

```bash
make build
```

### Testing

```bash
make test
make test-e2e
```

### Linting

```bash
make lint
```
